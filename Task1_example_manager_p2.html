The first task is to “decode” numbers into letters using a decoding key shown on the computer screen. Each correctly decoded number generates 100 output points for the firm. The more the employee decodes, the higher the firm’s output will be.

The decoding task will last for 5 minutes. The employee can continue decoding until the end of the period or choose to end the task earlier by selecting “END TASK” and pressing "Next" at any point in time.

The employee will be given 40 seconds to practice the task. This practice round does not generate any output points. It is solely for practice purposes.

You will wait in the meantime and read texts for 6 minutes. Once the time is over, You will learn the output that the employee has generated in the decoding task.
