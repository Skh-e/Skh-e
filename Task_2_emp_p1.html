Now, you will conduct an investment project for the firm. Recall that the project yields a revenue of 2200 points for the firm if it is realized but it entails costs. You will learn the actual costs on the next page. Then, you need to ask the manager for funding in order to realize the project.

The manager will not learn the actual costs; she/he only knows that the project costs can be any whole number between 1 and 2000 points, with all costs being equally likely.

So, you can request any number between the actual costs and 2000 points and the manager will have to approve your request. Next, the project will be realized and the firm will pay the manager the revenue minus the funds requested by you. If the requested funds are higher than the actual costs, you will keep the difference for yourself. The manager will never learn this difference.
