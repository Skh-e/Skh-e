The second task is to conduct an investment project for the firm. The project yields a revenue of 2200 points for the firm when it is realized. The project is always beneficial for the firm, but it is unclear how beneficial it is because it entails costs. Only you learn the actual costs. The manager does not learn the actual costs. The manager only knows that the project costs can be any whole number between 1 and 2000 points, with all costs being equally likely. You need to ask the manager for funding in order to realize the project. Specifically, you can request any number between the actual costs and 2000 points and the manager will have to approve your request. Next, the project will be realized and the firm will earn the revenue minus the funds requested by you and pay that amount to the manager. If the requested funds are higher than the actual costs, you will keep the difference for yourself. The manager will never learn this difference.
